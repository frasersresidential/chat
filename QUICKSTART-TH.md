# เริ่มใช้งาน OmniChat บน Windows (ฉบับมือใหม่)

ทำตามทีละขั้น ใช้เวลาประมาณ 5–10 นาที

---

## ขั้นที่ 1 — ติดตั้ง Node.js (ทำครั้งเดียว)

1. เข้าเว็บ 👉 https://nodejs.org
2. กดดาวน์โหลดปุ่มฝั่งซ้าย ที่เขียนว่า **LTS** (เวอร์ชันแนะนำ)
3. เปิดไฟล์ที่โหลดมา กด **Next** ไปเรื่อยๆ จนติดตั้งเสร็จ
4. เสร็จแล้ว **รีสตาร์ทเครื่อง** หนึ่งครั้ง (เพื่อให้ระบบรู้จัก Node)

---

## ขั้นที่ 2 — ดาวน์โหลดโค้ดแอป

**วิธีง่ายสุด (ไม่ต้องใช้ Git):**

1. เปิดลิงก์นี้:
   `https://github.com/frasersresidential/chat/archive/refs/heads/claude/relaxed-gauss-s9wp05.zip`
2. ไฟล์ ZIP จะถูกดาวน์โหลดมา
3. คลิกขวาที่ไฟล์ ZIP → **Extract All / แตกไฟล์ทั้งหมด**
4. จะได้โฟลเดอร์ชื่อ `chat-claude-relaxed-gauss-s9wp05`

> ถ้าคุณใช้ Git เป็นอยู่แล้ว ใช้คำสั่งนี้แทนได้:
> ```
> git clone https://github.com/frasersresidential/chat.git
> cd chat
> git checkout claude/relaxed-gauss-s9wp05
> ```

---

## ขั้นที่ 3 — รันแอป (ดับเบิลคลิก)

1. เปิดโฟลเดอร์ที่แตกไฟล์ไว้
2. ดับเบิลคลิกไฟล์ **`start-windows.bat`**
3. ครั้งแรกมันจะติดตั้งส่วนประกอบให้อัตโนมัติ (รอสักครู่)
4. พอขึ้นข้อความ `OmniChat running` เบราว์เซอร์จะเปิดหน้าแอปให้เอง
   หรือเปิดเองที่ 👉 **http://localhost:3000**

> ถ้า Windows เตือนความปลอดภัยตอนเปิด `.bat` ให้กด **More info → Run anyway**
> (เป็นไฟล์สคริปต์ของเราเอง ปลอดภัย)

---

## ขั้นที่ 4 — ลองเล่น

แอปมีข้อมูลตัวอย่างให้พร้อมเล่นทันที:

1. มุมขวาบน เลือก **act as** เป็น `Mia Manager` → จะเห็นแชตทั้งหมด (All inbox)
2. ไปแท็บ **Simulator** → เลือกช่องทาง เช่น *LINE OA Brand A* → พิมพ์ข้อความ → **Send inbound**
3. ดูแชตเด้งเข้า inbox และถูกจ่ายให้พนักงานขายอัตโนมัติ (round-robin)
4. ลองติ๊ก **VIP = Yes** → ระบบจะจ่ายให้ *Senior Sales*
5. ลองพิมพ์คำว่า `refund` หรือ `ร้องเรียน` → ระบบจ่ายให้ *Supervisor*
6. สลับ **act as** เป็นพนักงานที่ถูกจ่ายงาน เพื่อกดตอบลูกค้า

---

## หยุดแอป

กลับไปที่หน้าต่างสีดำ (Command Prompt) แล้วกด **Ctrl + C**

---

## เจอปัญหา?

| อาการ | วิธีแก้ |
|---|---|
| ดับเบิลคลิกแล้วบอกไม่พบ Node | ยังไม่ได้ติดตั้ง Node.js ที่ขั้น 1 หรือยังไม่รีสตาร์ทเครื่อง |
| เปิด http://localhost:3000 ไม่ขึ้น | ดูว่าหน้าต่างสีดำยังเปิดอยู่และขึ้นคำว่า `running` ไหม |
| พอร์ต 3000 ชนกับโปรแกรมอื่น | สร้างไฟล์ `.env` ใส่บรรทัด `PORT=4000` แล้วเปิด http://localhost:4000 |

ติดตรงไหนถามได้เลยครับ
